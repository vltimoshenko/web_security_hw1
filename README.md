## Как запустить

Требуются порты 8080, 8081, 5432
```
docker-compose up --build
```
Запросы проксируются через порт 
```
8080
```



## Работа с repeater-ом
Список сохранненых реквестов
```
http://localhost:8081/requests
```
Повтор реквеста
```
http://localhost:8081/<id>
```
